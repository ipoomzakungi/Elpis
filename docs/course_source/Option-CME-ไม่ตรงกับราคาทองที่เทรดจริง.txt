ทำไมราคา Option CME ไม่ตรงกับราคาทองที่เทรดจริง? และวิธีดริฟราคา GC → GO แบบมืออาชีพ
หนึ่งในปัญหาหลักของเทรดเดอร์ที่ใช้ข้อมูล Open Interest (OI) จาก CME มาวิเคราะห์ราคา
คือ ราคาใน CME ไม่ตรงกับราคาทองที่เราเทรดจริง ไม่ว่าจะเป็น GO, GOLDcash, XAUUSD หรือ GOLDspot ของโบรกต่าง ๆ
สิ่งนี้ทำให้ตีความ Call/Put OI Zone, 1SD Range, Volume/OI Ratio หรือแม้แต่การวาง Grid ผิดตำแหน่งได้ทันที
ดังนั้นจำเป็นต้องมีขั้นตอนหนึ่งที่ชื่อว่า
🟩 Price Drift (การดริฟราคา)
คือการ “แปลงราคา Strike ของ CME ให้ตรงกับราคาทองที่คุณเทรดจริง”
⸻
🟦 ทำไมราคา GC กับ GO ถึงไม่ตรงกัน?
เพราะ GC (CME Gold Futures) คือสัญญาฟิวเจอร์สแท้ มีโครงสร้างราคาแบบฟิวเจอร์ส:
• มี Rollover
• มี Contango/Backwardation
• มีต้นทุนการถือ (Cost of Carry)
แต่ GO ของโบรกเกอร์ไทย Tfex
ดังนั้นราคาจึง “ไม่เคยเท่ากัน” และ “ไม่ควรเอา Strike มาเทียบตรง ๆ”
⸻
🟩 ตัวอย่างจริง
• GC = 4052
• GO = 4085
ดังนั้นส่วนต่างราคาคือ
🟧 Offset = GO – GC = 4085 – 4052 = 33 ดอลลาร์
นี่คือ “ค่าส่วนต่างที่ต้องดริฟราคา”
เพื่อนำ Option Strike ทุกค่าใน CME ไปใช้กับ GO ได้ตรงจริง
⸻
🟦 วิธีใช้ Offset แบบถูกต้อง
ถ้า Option CME มี OI หนาแน่นที่ Strike ใดก็ตาม
เราต้อง ย้าย Strike ทั้งหมดขึ้น +33 ดอลลาร์
✔ ตัวอย่าง 1 — Call OI หนาแน่นที่ Strike 4050 (ใน CME)
แปลงเป็นราคาที่ใช้ใน GO
4050 + 33 = 4083
ดังนั้น Zone ที่ต้องระวังใน GO
คือราคาแถว 4083
⸻
✔ ตัวอย่าง 2 — Put OI หนาแน่นที่ Strike 4000
ใช้สูตรเดียวกัน
4000 + 33 = 4033
บนกราฟ GO
Zone Put จะอยู่ที่ 4033
⸻
✔ ตัวอย่าง 3 — โซน 1SD ของ CME อยู่ที่ 4060–4100
ใน GO ต้องดริฟทั้งกรอบ
• 4060 + 33 = 4093
• 4100 + 33 = 4133
ดังนั้นในตลาด GO
1SD ถูกต้องคือ 4093–4133 ไม่ใช่ 4060–4100
ถ้าไม่ดริฟราคา → การอ่าน OI พังทันที
และทำให้มุมมองผิดทั้งหมด
⸻
⸻
🟦 เคล็ดลับของเทรดเดอร์ระดับโปร
✔ ดริฟราคาใหม่วันละครั้ง ช่วงตลาดนิ่ง
✔ ใช้ GC Active Contract เท่านั้น
✔ ถ้าส่วนต่างคงที่ทั้งวัน → ใช้ Offset คงที่ 33
✔ ถ้าราคาหลุดกรอบ → อัปเดต Offset ใหม่ เพราะโบรกเกอร์บางเจ้าปรับ Mark-up ตามสภาพตลาด
✔ อย่าดู Strike แบบเทียบตรง ๆ เพราะ GO ไม่ใช่ GC
⸻
ถ้า GC = 4052 และ GO = 4085 → ทุก Strike ของ CME ต้อง +33 ดอลลาร์ เพื่อใช้กับ GO
นี่คือขั้นตอนพื้นฐานในการนำ OI Options CME มาใช้กับราคาทองที่เทรดจริง
ไม่ว่าคุณจะวาง Grid, วาง Zone, หรือคาดการณ์แรง Call/Put ก็ต้องผ่าน Price Drift ก่อนเสมอ
